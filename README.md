## What is GLOBAL Rest API?
GLOBAL Rest API is a solution proposed by me (Marcio Martins) as a resolution for a Recruiting Test subimited by GLOBAL to myself.

## About this document
This cover the most features about GLOBAL Rest API, including usage, error messages and known isses.

## How to install
GLOBAL Rest API runs in a Docker Container. It can be started by running `docker compose up -d`.

## How to check for API Documentation
Please refer to `https://localhost:8000/swagger` to check up API Documentation in OpenAPI format.
